# Node.js Training - 3 API

## Getting Started

### Prerequisites
- Node v6 or higher
- NPM

### Installing
```bash
$ npm install
```

### Running
```bash
$ npm start
````

```bash
$ npm run dev
```

```bash
$ npm test
```

### .env
```
# Server
PORT=3000

# Logger
LOG_LEVEL=debug
LOG_PRETTY_PRINT=true
```
