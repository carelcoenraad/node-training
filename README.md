## Requirements
- Node v6 or higher
- NPM
- Gulp

## Installing
- `$ npm i -g gulp`
- `$ npm i`

## Running
`$ gulp serve`

## Training information
### Functionality
- Project setup
  - ES6
  - Config
  - Gulp
  - ESLint
- Express server
  - Hello world
- Unit testing

- help (task-listing)

- pre-unit-test-report (Istanbul)
