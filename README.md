## Requirements
- Node v6 or higher
- NPM
- Gulp

## Installing
- `$ npm i -g gulp`
- `$ npm i`

## Running
`$ gulp serve`

## Training information
### Functionality
- Project setup
  - ES6
  - Config
  - Gulp
  - ESLint
- Express server
  - Hello world
  - Helmet middleware
  - Users API
  - PostgreSQL Database
  - Body-parser middleware
  - CRUD operations
  - Session
- Unit testing
- Static client-side

### Gulp tasks
- serve (Nodemon)
- lint (ESLint)
- help (task-listing)
- unit-test (Mocha)
- pre-unit-test-report (Istanbul)
- unit-test-report (Mocha + Istanbul)
