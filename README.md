# Node.js Training - 4 Database

## Getting Started

### Prerequisites
- Node v6 or higher
- NPM

### Installing
```bash
$ npm install
```

### Running
```bash
$ npm start
````

```bash
$ npm run dev
```

```bash
$ npm test
```

### .env
```
# Server
PORT=3000

# Database
DATABASE_CONNECTION_URI=postgres://username:password@host:port/database

# Logger
LOG_LEVEL=debug
LOG_PRETTY_PRINT=true
```
